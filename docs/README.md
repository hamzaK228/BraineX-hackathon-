## Getting Started

1. Open `index.html` in a web browser to view the website
2. All files are self-contained and work offline
3. Customize content, colors, and branding as needed

## Customization

### Colors
The main brand colors can be changed in `styles.css`:
- Primary gradient: `#667eea` to `#764ba2`
- Text colors: `#2d3748`, `#4a5568`, `#718096`

### Content
Update the following in `index.html`:
- Company name (currently "MentoraX")
- Contact information
- Mentor profiles
- Service descriptions
- Testimonials

### Images
Replace placeholder icons with actual photos:
- Add mentor photos
- Hero section illustration
- About section graphics
